# Halight Information Radiator AI - Live KPI Dashboards

## Project Overview
EdTech LMS for Halight: Real-time dashboards pulling KPIs (user engagement, client health) from Amazon Athena with ML trends/anomalies.

**Epic:** Information Radiator AI - Live KPI Dashboards. 
**Problem Solved:** Scattered KPIs delay decisions; enables shared real-time visibility.
